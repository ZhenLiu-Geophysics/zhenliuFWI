# zhenliuFWI

## What’s Coming

Get ready for a high-performance exploration of Liu Zhen's Full Waveform Inversion (FWI) algorithm. This repository will soon reveal a fresh, practical implementation designed to help you understand seismic inversion from the ground up.

## Why It Matters

FWI is one of the most powerful tools for turning raw seismic data into accurate subsurface models. This project aims to bring clarity, reproducibility, and real code for anyone who wants to level up their geophysics workflow.
